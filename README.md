# Mobile Web App

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)
[Live Demo](https://celadon-chaja-e416dc.netlify.app/)

A minimal **React + TypeScript + Vite** mobile web application template with **Tailwind CSS** and ESLint pre-configured. Designed for rapid development and easy deployment.

## Features

- React with **TypeScript**
- **Vite** for fast bundling and HMR
- **Tailwind CSS** for utility-first styling
- ESLint pre-configured for code quality
- Mobile-friendly responsive layout
- Ready to deploy to **Netlify** or other hosting services

---

## Tech Stack

- **Frontend:** React, TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS, PostCSS
- **Linting:** ESLint
- **Version Control:** Git / GitHub
